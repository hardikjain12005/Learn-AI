# Quiz App

An interactive **Quiz** application built to help users practice and test their knowledge in a simple and engaging way. It provides multiple questions, tracks score, and gives instant feedback on answers.

## Features

- Multiple-choice quiz questions with score tracking.  
- Shows correct and incorrect answers with instant feedback.  
- Simple and user-friendly interface for quick practice sessions.  
- Easily extendable question set and categories.
